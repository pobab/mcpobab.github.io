About Me

In 2015, I tried using Skript to make a feature but I didn't get what I wanted.

In 2019, I found Denizen and started using in until now.

Why im not using java?

I did used Java. I used Java for 3 years, I even have a spigot plugin there named Basic Adminshop. While I developed
my plugin, i found Denizen in earlier 2019 and decided to use it because it has almost all the spigot APIs I needed and it was easy to use.

So, why do I prefer to use Denizen rather than Java? Because it doesn't take that much longer to create a feature using Denizen rather than Java. The performance between Denizen and Java are also not so different. And from my 4 years experience of using Denizen, I can guarantee the performance is very excellent.
